## ◈ Featured Projects

<details>
<summary><strong>⬡ BIC Campus Placement System</strong> — Centralised College Placement Backend</summary>
<br>

A centralised placement management platform enabling students to browse and apply for job listings while tracking placement status in real time. Administrators manage company partnerships and student pipelines from a dedicated dashboard.

| Attribute | Detail |
|:---|:---|
| **Stack** | Python · Django · Django REST Framework · PostgreSQL · JWT |
| **Architecture** | Role-based multi-user system (Student / Company / Admin) |
| **Auth Layer** | JWT authentication with token refresh and role enforcement |
| **API Design** | Custom ViewSets, Serializers, Nested Endpoints, Filtering |
| **Security** | RBAC Permission Classes, Protected Routes, Validated Payloads |
| **Impact** | Deployed for a live college cohort — streamlines placement for 500+ students |

Built the complete backend from schema design through API delivery. Implemented custom permission guards ensuring students, companies, and administrators operate within strictly scoped access boundaries. All API contracts are documented and consumed by a React.js frontend.

</details>
